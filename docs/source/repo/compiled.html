如何編譯

  1. 在檔案 AON.uproject 右鍵，選擇 Switch Unreal Engine Version，接著選擇 4.19。

  2. 等待步驟一結束後，目錄會產生專案檔 AON.sln，直接點選開啟。

  3. 進入 VS 後，請壓快速鍵 Ctrl+F5 進行編譯。

    請注意右側專案資訊 VS 版本是否為 2017。

  4. 編譯成功後將自動開啟 Unreal。

    如果失敗，請確認第三方套件是否安裝、專案是否更新成功。
